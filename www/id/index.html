Cara Kerja

Dompet + Gateway = tumpukan mikropembayaran

Agen AI mengakses API yang dilindungi, menerima faktur 402, membayar on-chain dengan ClawPurse, dan mencoba lagi dengan bukti. Gateway memverifikasi transaksi di Neutaro dan meneruskan ke upstream. Tanpa akun, tanpa kunci API — hanya penyelesaian on-chain.

Dompet

Fitur Dompet ClawPurse

Semua yang Anda butuhkan untuk mengelola NTMPI secara lokal — dengan pagar pengaman yang dirancang untuk agen otonom.

Keystore Terenkripsi

AES-256-GCM dengan derivasi kunci scrypt. Mnemonik tetap terkunci di node lokal.

Wizard Pagar Pengaman

Pengaturan interaktif untuk batas pengiriman, ambang konfirmasi, dan mode penegakan daftar izin.

Daftar Tujuan yang Diizinkan

Terapkan atau dokumentasikan alamat tepercaya dengan batas per-tujuan, memo wajib, dan catatan.

Tanda Terima Transaksi

Setiap pengiriman dicatat secara lokal dengan hash tx, tinggi blok, memo, dan status pagar pengaman untuk jejak audit lengkap.

Staking & Delegasi

Delegasikan, batalkan delegasi, re-delegasi NTMPI. Temukan validator, lacak hadiah, dan kelola periode unbonding 22 hari.

API Pemrograman

Impor fungsi dompet langsung ke agen dan skrip. send(), getBalance(), delegate() dan lainnya.

Gateway 402

Monetisasi API apa pun dengan mikropembayaran NTMPI

ClawPurse 402 Gateway berada di depan API upstream Anda dan menangani penagihan, verifikasi on-chain, dan proxy. Dua alur pembayaran, penetapan harga per-rute, dan API manajemen yang dilindungi JWT.

Bayar-per-permintaan

Klien menerima faktur 402, membayar on-chain, mencoba lagi dengan header X-Payment-Proof. Gateway memverifikasi dan meneruskan.

Saldo prabayar

Klien menyetor NTMPI sekali. Permintaan berikutnya menggunakan X-Client-Id dan gateway memotong per panggilan — tanpa transaksi on-chain berulang.

Verifikasi on-chain

Mengkueri Neutaro REST API untuk mengonfirmasi jumlah pembayaran, memo, dan konfirmasi blok.

Penetapan harga per-rute

Harga default per permintaan plus override per-rute. Pembacaan murah, penulisan mahal.

Deploy Docker & bare-metal

Satu perintah docker compose up atau skrip deploy Ubuntu. Penyimpanan SQLite, tanpa dependensi eksternal.

Referensi API lengkap, panduan konfigurasi, dan contoh integrasi agen di gateway.clawpurse.ai

Integrasi Agen

Dukungan kelas satu untuk AI agentik

npm install && npm run build && npm link

# Buat dompet

clawpurse address
clawpurse balance --password <pass>

# Kirim token
clawpurse send neutaro1... 1.25 --password <pass>

# Staking (v2.0)
clawpurse validators
clawpurse stake neutarovaloper1... 100 --password <pass>
clawpurse delegations
Ekosistem

Didukung oleh Timpi & blockchain Neutaro

ClawPurse adalah lapisan pembayaran untuk jaringan pencarian independen Timpi. Timpi menggabungkan pencarian yang mengutamakan privasi, API Data terbuka, dan node DePIN yang memberi hadiah kepada operator yang menjalankan jaringan TAP.

Stack Timpi

Pencarian, Data, DePIN

  • Mesin Pencari: hasil tahan sensor tanpa pembuatan profil atau pelacakan.
  • API Data: akses pemrograman ke indeks terdesentralisasi.
  • Node DePIN: Collector, Guardian, dan GeoCore yang mendapatkan NTMPI.

Bergabung di timpi.io.

Cara mendapatkan NTMPI

Pasangan Spot MEXC

  1. Masuk ke MEXC dan setor USDT.
  2. Buka pasar spot NTMPI/USDT.
  3. Buat pesanan, lalu tarik ke alamat Neutaro Anda.

Gravity Bridge → Osmosis

  1. Bridge stablecoin dari Ethereum melalui Gravity Pulse.
  2. IBC-hop ke Osmosis melalui Keplr atau Leap.
  3. Tukar ke NTMPI dan IBC ke Neutaro.
