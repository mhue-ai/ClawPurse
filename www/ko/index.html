작동 방식

지갑 + 게이트웨이 = 마이크로페이먼트 스택

AI 에이전트가 게이트된 API에 접근하면 402 인보이스를 받고, ClawPurse로 온체인 결제한 후 증명과 함께 재시도합니다. 게이트웨이가 Neutaro에서 트랜잭션을 검증하고 업스트림으로 프록시합니다. 계정 불필요, API 키 불필요 — 온체인 결제만.

  AI 에이전트 / 스크립트 / 브라우저
    │
    ├── GET /api/data ───────────────────▶  402 게이트웨이
    │                                            │
    │   ◀── 402 Payment Required ──────────────┘
    │       { invoiceId, amount, address, memo }
    │
    ├── clawpurse send   --memo 
    │       (지갑을 통한 온체인 NTMPI 결제)
    │
    ├── GET /api/data ───────────────────▶  402 게이트웨이
    │   Header: X-Payment-Proof:    │
    │                                            │
    │   게이트웨이가 Neutaro 체인에서 tx 검증  │
    │                                            │
    │   ◀── 200 OK (업스트림에서 프록시) ──┘
지갑

ClawPurse 지갑 기능

NTMPI를 로컬에서 관리하는 데 필요한 모든 것 — 자율 에이전트를 위한 가드레일 내장.

암호화 키스토어

scrypt 키 유도와 AES-256-GCM. 니모닉은 로컬 노드에 잠겨 있습니다.

가드레일 마법사

송금 한도, 확인 임계값, 허용 목록 강제 모드의 대화형 설정.

대상 허용 목록

대상별 한도, 필수 메모, 메모와 함께 신뢰할 수 있는 주소를 강제하거나 기록.

트랜잭션 영수증

모든 송금을 tx 해시, 블록 높이, 메모, 가드레일 상태와 함께 로컬에 기록하여 완전한 감사 추적 제공.

스테이킹 및 위임

NTMPI 위임, 위임 해제, 재위임. 검증자 탐색, 보상 추적, 22일 언본딩 기간 관리.

프로그래밍 API

지갑 함수를 에이전트와 스크립트에 직접 가져오기. send(), getBalance(), delegate() 등.

402 게이트웨이

NTMPI 마이크로페이먼트로 모든 API 수익화

ClawPurse 402 게이트웨이는 업스트림 API 앞에 위치하며 인보이스 발행, 온체인 검증, 프록시를 처리합니다. 2가지 결제 흐름, 라우트별 가격 설정, JWT 보호 관리 API.

요청별 결제

클라이언트가 402 인보이스를 받고, 온체인으로 결제하고, X-Payment-Proof 헤더와 함께 재시도. 게이트웨이가 검증하고 프록시.

선불 잔액

클라이언트가 NTMPI를 한 번 입금. 이후 요청은 X-Client-Id를 사용하고 게이트웨이가 호출당 차감 — 반복적인 온체인 거래 불필요.

온체인 검증

Neutaro REST API에 쿼리하여 결제 금액, 메모, 블록 확인을 확인.

라우트별 가격 설정

기본 요청 단가 + 라우트별 오버라이드. 저렴한 읽기, 비싼 쓰기.

Docker 및 베어메탈 배포

docker compose up 한 명령 또는 Ubuntu 배포 스크립트. SQLite 스토리지, 외부 종속성 없음.

전체 API 레퍼런스, 설정 가이드, 에이전트 통합 예제는 여기에서 gateway.clawpurse.ai

CAPTCHA 대신 PoW

자율 402 흐름

에이전트가 402 응답을 감지하고, 인보이스를 파싱하고, 온체인으로 결제하고, 증명과 함께 재시도 — 모두 프로그래밍 방식으로.

빠른 시작

1분 안에 실행

# 지갑 설치
npm install && npm run build && npm link

# 지갑 생성
clawpurse init --password <pass>
clawpurse address
clawpurse balance --password <pass>

# 토큰 전송
clawpurse send neutaro1... 1.25 --password <pass>

# 스테이킹 (v2.0)
clawpurse validators
clawpurse stake neutarovaloper1... 100 --password <pass>
clawpurse delegations
생태계

Timpi와 Neutaro 블록체인으로 구동

ClawPurse는 Timpi의 독립 검색 네트워크를 위한 결제 레이어입니다. Timpi는 프라이버시 우선 검색, 오픈 데이터 API, TAP 네트워크 운영자에게 보상하는 DePIN 노드를 결합합니다.

Timpi 스택

검색, 데이터, DePIN

  • 검색 엔진: 프로파일링이나 추적 없는 검열 저항 결과.
  • 데이터 API: 분산 인덱스에 대한 프로그래밍 접근.
  • DePIN 노드: NTMPI를 획득하는 Collectors, Guardians, GeoCores.
