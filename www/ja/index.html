ゲートウェイ

ClawPurse 402 ゲートウェイ

402 Payment Required を実装する HTTP リバースプロキシ。リクエスト単位課金、前払い残高、オンチェーン検証、ルートレベル価格設定で任意の API を NTMPI マイクロペイメントでゲーティング。

仕組み

ウォレット + ゲートウェイ = マイクロペイメントスタック

AI エージェントがゲート付き API にアクセスし、402 インボイスを受け取り、ClawPurse でオンチェーン決済し、証明付きでリトライします。ゲートウェイが Neutaro でトランザクションを検証し、アップストリームへプロキシします。アカウント不要、API キー不要 — オンチェーン決済のみ。

  AI エージェント / スクリプト / ブラウザ
    │
    ├── GET /api/data ───────────────────▶  402 ゲートウェイ
    │                                            │
    │   ◀── 402 Payment Required ──────────────┘
    │       { invoiceId, amount, address, memo }
    │
    ├── clawpurse send   --memo 
    │       （ウォレット経由のオンチェーン NTMPI 決済）
    │
    ├── GET /api/data ───────────────────▶  402 ゲートウェイ
    │   Header: X-Payment-Proof:    │
    │                                            │
    │   ゲートウェイが Neutaro チェーンで tx を検証  │
    │                                            │
    │   ◀── 200 OK（アップストリームからプロキシ） ──┘
ウォレット

ClawPurse ウォレット機能

NTMPI をローカルで管理するために必要なすべて — 自律エージェント向けのガードレール搭載。

暗号化キーストア

scrypt 鍵導出を使用した AES-256-GCM。ニーモニックはローカルノードにロックされます。

ガードレールウィザード

送金制限、確認しきい値、許可リスト強制モードのインタラクティブセットアップ。

送金先許可リスト

送金先ごとの上限、必須メモ、メモ付きで信頼されたアドレスを強制または記録。

トランザクション領収書

すべての送金を tx ハッシュ、ブロック高、メモ、ガードレールステータスとともにローカルに記録し、完全な監査証跡を実現。

ステーキングと委任

NTMPI の委任、委任解除、再委任。バリデーター発見、報酬追跡、22 日間のアンボンディング期間管理。

プログラマティック API

ウォレット関数をエージェントやスクリプトに直接インポート。send()、getBalance()、delegate() など。

402 ゲートウェイ

NTMPI マイクロペイメントで任意の API を収益化

ClawPurse 402 ゲートウェイはアップストリーム API の前に配置され、インボイス発行、オンチェーン検証、プロキシを処理します。2 つの決済フロー、ルートレベル価格設定、JWT 保護の管理 API。

リクエスト単位課金

クライアントが 402 インボイスを受け取り、オンチェーンで決済し、X-Payment-Proof ヘッダー付きでリトライ。ゲートウェイが検証してプロキシ。

前払い残高

クライアントが一度 NTMPI を入金。以降のリクエストは X-Client-Id を使用し、ゲートウェイがコールごとに差し引き — オンチェーン取引の繰り返し不要。

オンチェーン検証

Neutaro REST API に問い合わせて支払い金額、メモ、ブロック確認を確認。

ルートレベル価格設定

デフォルトのリクエスト単価とルートごとのオーバーライド。安い読み取り、高い書き込み。

Docker & ベアメタルデプロイ

docker compose up のワンコマンドまたは Ubuntu デプロイスクリプト。SQLite ストレージ、外部依存なし。

完全な API リファレンス、設定ガイド、エージェント統合例はこちら gateway.clawpurse.ai

エコシステム

Timpi と Neutaro ブロックチェーンによる基盤

ClawPurse は Timpi の独立検索ネットワークのペイメントレイヤーです。Timpi はプライバシーファーストの検索、オープンデータ API、TAP ネットワーク運営者に報酬を与える DePIN ノードを組み合わせています。

Timpi スタック

検索、データ、DePIN

  • 検索エンジン：プロファイリングやトラッキングなしの検閲耐性結果。
  • データ API：分散型インデックスへのプログラマティックアクセス。
  • DePIN ノード：NTMPI を獲得する Collectors、Guardians、GeoCores。

参加はこちら timpi.io.

NTMPI の入手方法

MEXC スポットペア

  1. MEXC にサインインして USDT を入金。
  2. NTMPI/USDT スポット市場を開く。
  3. 注文を出し、Neutaro アドレスに出金。

Gravity Bridge → Osmosis

  1. Gravity Pulse 経由で Ethereum からステーブルコインをブリッジ。
  2. Keplr または Leap 経由で Osmosis に IBC ホップ。
  3. NTMPI にスワップして Neutaro に IBC 送金。
