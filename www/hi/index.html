वॉलेट + गेटवे = माइक्रोपेमेंट स्टैक

एक AI एजेंट गेटेड API को हिट करता है, 402 इनवॉइस प्राप्त करता है, ClawPurse से ऑन-चेन भुगतान करता है, और प्रूफ़ के साथ पुनः प्रयास करता है। गेटवे Neutaro पर ट्रांज़ैक्शन सत्यापित करता है और अपस्ट्रीम को प्रॉक्सी करता है। कोई अकाउंट नहीं, कोई API कुंजी नहीं — केवल ऑन-चेन सेटलमेंट।

  AI एजेंट / स्क्रिप्ट / ब्राउज़र
    │
    ├── GET /api/data ───────────────────▶  402 गेटवे

    │       { invoiceId, amount, address, memo }

    ├── clawpurse send   --memo

    │

    │   Header: X-Payment-Proof:    │

    │                                            │

वॉलेट

ClawPurse वॉलेट फ़ीचर्स

NTMPI को लोकली मैनेज करने के लिए सब कुछ — ऑटोनोमस एजेंट के लिए गार्डरेल इनबिल्ट।

एन्क्रिप्टेड कीस्टोर

scrypt की डेरिवेशन के साथ AES-256-GCM। निमोनिक्स लोकल नोड पर लॉक रहते हैं।

गार्डरेल विज़ार्ड

भेजने की सीमा, पुष्टि थ्रेशहोल्ड और अलाउलिस्ट एनफ़ोर्समेंट मोड का इंटरैक्टिव सेटअप।

डेस्टिनेशन अलाउलिस्ट

प्रति-डेस्टिनेशन कैप, आवश्यक मेमो और नोट्स के साथ विश्वसनीय पतों को एनफ़ोर्स या डॉक्यूमेंट करें।

ट्रांज़ैक्शन रसीदें

हर भेजा गया tx हैश, ब्लॉक हाइट, मेमो और गार्डरेल स्टेटस के साथ लोकली रिकॉर्ड — पूर्ण ऑडिट ट्रेल के लिए।

स्टेकिंग और डेलिगेशन

NTMPI डेलिगेट, अनडेलिगेट, रीडेलिगेट करें। वैलिडेटर खोजें, रिवॉर्ड ट्रैक करें और 22-दिन की अनबॉन्डिंग अवधि प्रबंधित करें।

प्रोग्रामैटिक API

वॉलेट फ़ंक्शन सीधे एजेंट और स्क्रिप्ट में इम्पोर्ट करें। send(), getBalance(), delegate() और अधिक।

402 गेटवे

NTMPI माइक्रोपेमेंट से किसी भी API को मोनेटाइज़ करें

ClawPurse 402 गेटवे आपकी अपस्ट्रीम API के सामने बैठता है और इनवॉइसिंग, ऑन-चेन वेरिफ़िकेशन और प्रॉक्सीइंग को हैंडल करता है। दो पेमेंट फ़्लो, रूट-लेवल प्राइसिंग और JWT-प्रोटेक्टेड मैनेजमेंट API।

पे-पर-रिक्वेस्ट

क्लाइंट 402 इनवॉइस प्राप्त करता है, ऑन-चेन भुगतान करता है, X-Payment-Proof हेडर के साथ पुनः प्रयास करता है। गेटवे सत्यापित करता है और प्रॉक्सी करता है।

प्रीपेड बैलेंस

क्लाइंट एक बार NTMPI जमा करता है। बाद की रिक्वेस्ट X-Client-Id का उपयोग करती हैं और गेटवे प्रति कॉल कटौती करता है — बार-बार ऑन-चेन ट्रांज़ैक्शन नहीं।

ऑन-चेन वेरिफ़िकेशन

भुगतान राशि, मेमो और ब्लॉक कन्फ़र्मेशन सत्यापित करने के लिए Neutaro REST API से क्वेरी।

रूट-लेवल प्राइसिंग

डिफ़ॉल्ट प्रति-रिक्वेस्ट कीमत प्लस प्रति-रूट ओवरराइड। सस्ती रीड, महंगी राइट।

Docker और बेयर-मेटल डिप्लॉय

एक कमांड docker compose up या Ubuntu डिप्लॉय स्क्रिप्ट। SQLite स्टोरेज, ज़ीरो एक्सटर्नल डिपेंडेंसी।

पूर्ण API रेफ़रेंस, कॉन्फ़िगरेशन गाइड और एजेंट इंटीग्रेशन उदाहरण यहाँ gateway.clawpurse.ai

Timpi और Neutaro ब्लॉकचेन द्वारा संचालित

ClawPurse Timpi के स्वतंत्र सर्च नेटवर्क की पेमेंट लेयर है। Timpi प्राइवेसी-फ़र्स्ट सर्च, ओपन डेटा API और TAP नेटवर्क चलाने वाले ऑपरेटर को रिवॉर्ड देने वाले DePIN नोड्स को जोड़ता है।

Timpi स्टैक

सर्च, डेटा, DePIN

  • सर्च इंजन: बिना प्रोफ़ाइलिंग या ट्रैकिंग के सेंसरशिप-रेज़िस्टेंट परिणाम।
  • डेटा API: डीसेंट्रलाइज़्ड इंडेक्स तक प्रोग्रामैटिक एक्सेस।
  • DePIN नोड्स: NTMPI कमाने वाले Collectors, Guardians और GeoCores।
