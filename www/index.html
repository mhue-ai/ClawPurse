ClawPurse Beta

A local-only Timpi / NTMPI wallet for OpenClaw nodes. Generate, guard, and verify transactions with encrypted keystores, destination allowlists, and auditable receipts — no third-party custody required.

View on GitHub Read the docs

Key Features

  • Encrypted Keystore

    AES-256-GCM with scrypt key derivation keeps mnemonics locked to the local node.

  • Guardrail Wizard

    Interactive setup for send limits, confirmation thresholds, and allowlist enforcement.

  • Destination Allowlists

    Enforce or document trusted addresses with per-destination caps, memos, and notes.

  • Transaction Receipts

    Every send is recorded locally with tx hash, block height, memo, and guardrail status.

  • Programmatic API

    Import ClawPurse functions in agents or scripts for automated payouts and balance checks.

  • OpenClaw Skill

    Drop-in SKILL.md lets other nodes install ClawPurse for trust-minimized payments.

CLI Quick Start

npm install
npm run build
npm link

clawpurse init --password <pass>
clawpurse address
clawpurse balance --password <pass>
clawpurse send neutaro1... 1.25 --password <pass>
