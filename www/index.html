Where to get TIMPI / NTMPI

MEXC Spot Pair

  1. Sign in to MEXC and deposit USDT (TRC-20 or BEP-20 is fastest).
  2. Open the NTMPI/USDT spot market.
  3. Place a limit or market order, then withdraw to your Neutaro / ClawPurse address.

Gravity Bridge → Osmosis

  1. Bridge USDC/USDT from Ethereum using Gravity Pulse or Blockscape.
  2. IBC-hop the bridged stablecoin to Osmosis via Keplr or Leap.
  3. Swap for NTMPI on Osmosis, then IBC the funds to Neutaro and import them into ClawPurse.

Need the detailed playbook? See HOWTOGETTIMPI.md.

Version 2.0 Highlights

Version 2.0.0 adds full staking support to the local wallet stack:

  • Staking CLI: stake, unstake, redelegate, delegations, validators, unbonding
  • Staking API: delegate(), undelegate(), redelegate(), getDelegations(), getValidators()
  • 22-day unbonding: Full support for Neutaro's unbonding period with clear warnings
  • Validator discovery: List all active validators with commission rates
  • All v1.0 features: encrypted keystore, send/receive, allowlists, receipts, guardrails

Version 2.0.1 Enhancements

Version 2.0.1 adds enterprise-grade security and testing infrastructure:

  • Security Layer: Password strength enforcement (12+ chars), comprehensive input validation for addresses, amounts, and memos
  • Test Coverage: 60+ automated tests across unit, integration, and E2E suites with CI/CD pipeline
  • Enhanced API: Security validation utilities (validatePassword(), validateAddress(), validateAmount())
  • Developer Tools: Jest configuration, GitHub Actions workflow, comprehensive test documentation
  • Documentation: 6 new guides including test plan, security improvements, and quick start
  • All v2.0 features: staking, validators, delegations, plus enhanced security and reliability
